Studying HTML
  • <!DOCTYPE html>
    - html 문서 작성시 가장 윗줄에 보편적으로 작성하며 현 문서가 HTML임을 알려주는 부분

  • <html> '내용' </html>
    - !DOCTYPE html 작성후 뒤이어 작성하며 '내용' 부분에 head 및 body가 들어간다.

  • <head> '내용' </head>
    - 이 부분은 HTML 문서에 관한 기본 정보를 포함하는 부분으로 주로 문서의 제목, 제작자 문서정보 등이 포함된다.
       title, meta, script, style 태그 등등이 이에 해당된다.

  • <title> '내용' </title>
    - 현 페이지의 제목을 설정하는 부분으로 '내용'부분에 적힌 텍스트가 웹브라우저의 탭란에 표시된다.

  • <meta> '내용' </meta>
    - <meta>태그는 해당 문서에 대한 정보인 메타데이터(metadata)를 정의할때 사용된다.
       base, link, script, style, title등의 태그와 같은 다른 메타데이터 관련 요소들이 나타낼 수 없는 다양한 종류의 메타데이터를 제공할 때 사용되며, 이렇게 제공된 정보는 브라우저나 검색 엔진, 다른 웹 서비스에서 사용하게 된다.
       <meta>태그는 항상 <head> 내부에 위치해야 한다.
       <meta charset="utf-8">로 보통 사용 되며 이 태그는 html파일의 인코딩을 알려주는 태그이다. 참고로 utf-8은 유니코드를 위한 가변 길이 문자 인코딩 방식 중 하나이다.

  • <body> '내용' </body>
    - <body>태그는 HTML 문서의 본문에 해당되며 실제 화면에 나타는 내용을 주로 기술한다. 즉 사용자가 웹트라우저 상 표현하고자 하는 내용들을 다양한 태그를 사용하여 구성하는 부분으로 웹문서의 대부분을 차지한다.
